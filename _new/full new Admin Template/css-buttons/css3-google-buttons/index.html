CSS3 Google Buttons

CSS3 Google Buttons helps you easily create Google-style UI elements.

Buttons

The "buttons" can be created by adding class="button" to any appropriate <a>, <button>, or <input> element. 

<a href="#" class="button">Post comment</a>
New Comment (link)

Colored Buttons

Red and blue colored buttons can be created by adding the red, blue, or green class to your button. 

<a href="#" class="button red">Compose Mail</a>
Compose Mail Search Mail Add Contact

Disabled Buttons

Buttons can also be disabled. The traditional style used in Google Calendar displays the button with a white background and greyed out text. Add the disabled class for this style. If you'd rather keep the grey background but dim it, you can add the disabled2 class. 

<a href="#" class="button disabled">Today</a>
<a href="#" class="button disabled2">Today</a>
Today Today

Smaller Buttons

Default buttons too big? Shrink the text-size of the button with the sm class, or the size of the entire button with the mini class. 

<a href="#" class="button sm">Month</a>
<a href="#" class="button mini">Month</a>
Month Month

Grouped buttons

Groups of buttons can be created by wrapping them in an element given a class of button-group. 

<div class="button-group">
    <a href="#" class="button">Archive</a>
    <a href="#" class="button">Spam</a>
    <a href="#" class="button">Delete</a>
</div>
Archive Spam Delete

Mixed groups

Displaying a mixture of grouped and standalone buttons, as might be seen in a toolbar, can be done by adding another wrapping element with the class button-container. 

<div class="actions button-container">
    <a href="#" class="button">Compose New</a>
    
    <div class="button-group">
        <a href="#" class="button">Archive</a>
        <a href="#" class="button">Spam</a>
        <a href="#" class="button">Delete</a>
    </div>
                        
    <div class="button-group minor-group">
        <a href="#" class="button">Move to</a>
        <a href="#" class="button">Labels</a>
    </div>

    <a href="#" class="button">More</a>
</div>

Radio Button Groups

Sometimes your button group should act like a series of radio buttons, in which selecting one button deselects the others. Add the toggle-buttons class to your button-group div to accomplish this. To set the currently selected button, use the checked class. 

<div class="button-group toggle-buttons">
    <a href="#" class="button">Day</a>
    <a href="#" class="button">Week</a>
    <a href="#" class="button checked">Month</a>
    <a href="#" class="button">4 Days</a>
    <a href="#" class="button">Agenda</a>
</div>
Day Week Month 4 Days Agenda 
<div class="button-group toggle-buttons">
    <a href="#" class="button mini">Day</a>
    <a href="#" class="button mini">Week</a>
    <a href="#" class="button mini checked">Month</a>
    <a href="#" class="button mini">4 Days</a>
    <a href="#" class="button mini">Agenda</a>
</div>

Buttons with icons only

Create a button with only an icon (no text) by adding the icon no-text classes, along with one of the icon classes below. 


<a href="#" class="button icon return no-text"></a>
<a href="#" class="button icon reload no-text"></a>

Or group them together, just like Gmail's paginator.

<div class="button-group">
    <a href="#" class="button icon rewind no-text"></a>
    <a href="#" class="button icon play no-text"></a>
<div>

You can even mix and match with the disabled state.

<div class="button-group">
    <a href="#" class="button icon rewind no-text disabled2"></a>
    <a href="#" class="button icon play no-text"></a>
<div>

Mini paginator buttons

The new back/forward buttons inside input elements on Google Flights is pretty slick. You can create those too by adding the xs pg bw and xs pg fw classes. 

<div class="button-group">
    <a href="#" class="button xs pg bw"></a>
    <a href="#" class="button xs pg fw"></a>
<div>

Buttons with text and icon

A range of icons can be added (only for links and buttons) by adding a class of icon and any one of the provided icon classes below. 

<div class="button-group">
    <a href="#" class="button xs pg bw"></a>
    <a href="#" class="button xs pg fw"></a>
<div>

Menu Buttons

Create buttons with dropdown menus, just like Google Docs, easily. Of course, it will require a little bit of Javascript magic, but the CSS is all here.

Add the icon menu classes to your button. The Javascript used here also requires a data-menu attribute to be set on your button. The value should be the id of the UL for your menu. 

<a class="button icon menu" data-menu="btn-menu">Menu</a>
<a class="button icon menu" data-menu="btn-menu">Another Menu</a>

<ul id="btn-menu" class="menu">
    <li><a href="">Document</a></li>
    <li><a href="">Presentation</a></li>
    <li><a href="">Spreadsheet</a></li>
    <li><a href="">Form</a></li>
</ul>
